Keep all the training images within their sub-folders.

Example:

* train:
  * sunflower:
    * image_1.jpg
    * image_2.jpg
    ...
    ...
  * tulip:
    * image_1.jpg
    * image_2.jpg
    ...
    ...

Delete this file once you have the above folder structure.
(Full set of images are vailable at http://www.robots.ox.ac.uk/~vgg/data/flowers/17/)
